# web3-drip-tracker

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)

An open-source dashboard template and indexing tool for tracking streaming finance, token vesting schedules, and recurring crypto payments.

## Features
- Fully typed and robust implementation
- Highly customizable and modular
- Built for the modern Web3 ecosystem
